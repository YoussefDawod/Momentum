# Momentum

Eleganter Pomodoro-Timer. Fokussiert arbeiten, strukturiert pausieren, Fortschritt sehen.

**Live:** [momentum.yellowdeveloper.de](https://momentum.yellowdeveloper.de)

## Features

- Drift-freier Timer (Wall-Clock-basiert)
- Vier Methoden: Klassisch, Tiefenfokus, Sprint, Benutzerdefiniert
- Statistik mit Streak, Export und wöchentlicher Übersicht
- Hell/Dunkel/System-Theme
- Tastatur-Shortcuts, PWA-installierbar, offline-fähig
- Keine Tracker, alle Daten lokal im Browser

## Setup

```bash
npm install
npm run dev       # http://localhost:5173
npm run build
npm test
```
